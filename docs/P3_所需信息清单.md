# P3 开发所需信息清单

请在 .env 中提供以下环境变量（无需在文档中写真实值）：

## 1. 模型与向量化（LlamaIndex 相关）

- `LLM_PROVIDER`：模型提供方类型（例如 openai_compatible / qwen / azure_openai / other）
- `LLM_API_BASE`：模型 API 基础地址
- `LLM_API_KEY`：模型 API Key
- `LLM_MODEL`：聊天模型名称

- `EMBEDDING_PROVIDER`：向量化提供方类型
- `EMBEDDING_API_BASE`：向量化 API 基础地址
- `EMBEDDING_API_KEY`：向量化 API Key
- `EMBEDDING_MODEL`：向量化模型名称

## 2. 向量库与数据库

- `DATABASE_URL`：PostgreSQL 连接串（含 pgvector）
- `PGVECTOR_TABLE`：向量表名（如 chunks 或 llamaindex_nodes）
- `PGVECTOR_SCHEMA`：向量表 schema（默认 public）

## 3. 对象存储（MinIO）

- `MINIO_ENDPOINT`
- `MINIO_ACCESS_KEY`
- `MINIO_SECRET_KEY`
- `MINIO_BUCKET`
- `MINIO_SECURE`：true / false

## 4. 异步任务（Celery / Redis）

- `REDIS_URL`
- `CELERY_BROKER_URL`
- `CELERY_RESULT_BACKEND`

## 5. 文档入库参数（可选）

- `INGEST_BATCH_SIZE`：向量化批大小
- `INGEST_MAX_FILE_MB`：单文件大小限制
- `INGEST_CHUNK_SIZE`：切片长度
- `INGEST_CHUNK_OVERLAP`：切片重叠

## 6. 其他（可选）

- `APP_ENV`：dev / prod
- `LOG_LEVEL`：info / debug
