# 一键启动/停止（Windows 开发）

本文档提供 Windows + PowerShell 的一键启动/停止方式，适用于本地开发。

## 启动（一次性命令）

在项目根目录执行：

```powershell
if (!(Test-Path .env)) { Copy-Item .env.example .env }
docker compose --env-file .env -f deploy/docker-compose.dev.yml up -d

Start-Process powershell -ArgumentList "-NoExit", "-Command", "Set-Location $PWD\\backend; .\\.venv\\Scripts\\Activate.ps1; pip install -e '.[dev]'; alembic upgrade head; uvicorn app.main:app --reload --host 127.0.0.1 --port 8000"

Start-Process powershell -ArgumentList "-NoExit", "-Command", "Set-Location $PWD\\frontend; npm install; npm run dev"
```

说明：
- 第一次运行会创建 `.env` 并启动 PostgreSQL/Redis/MinIO。
- 后端会自动安装依赖并执行数据库迁移。
- 前端会自动安装依赖并启动 Vite。
- 两个 `Start-Process` 会打开新终端窗口（保持可见日志）。

## 停止（一次性命令）

在项目根目录执行：

```powershell
docker compose --env-file .env -f deploy/docker-compose.dev.yml down
```

说明：
- 前后端进程请在各自终端窗口按 `Ctrl+C` 停止。
- 如需强制关闭，可在任务管理器结束 `uvicorn`/`node` 进程。

## 访问地址

- 后端健康检查：`http://127.0.0.1:8000/health`
- 后端数据库健康检查：`http://127.0.0.1:8000/health/db`
- 前端页面：`http://127.0.0.1:5173`
